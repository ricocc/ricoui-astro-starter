# Claude Theme

Claude is a warm vellum color theme. It uses creamy paper surfaces, articulate ink text, restrained borders, and a small terra cotta accent.

## Runtime

- Theme id: `claude`
- Applied with: `<html data-theme="claude">`
- Stored in: `localStorage.theme_id`
- Dark mode is layered with: `<html class="dark" data-theme="claude">`

## Light Tokens

| Token | Value | Usage |
| --- | --- | --- |
| `--color-primary` | `#141413` | Brand color, headings, links, icons |
| `--color-primary-strong` | `#1f1e1d` | Hover and emphasis |
| `--color-primary-light` | `#9c9a92` | Soft accents and muted UI |
| `--color-accent` | `#d97757` | Terra cotta accent moments |
| `--color-bg-primary` | `#faf9f5` | Page canvas |
| `--color-bg-secondary` | `#ffffff` | Cards and panels |
| `--color-text-primary` | `#141413` | Heading color |
| `--color-text-secondary` | `#3d3d3a` | Body text |
| `--color-text-tertiary` | `#73726c` | Muted metadata |
| `--color-btn-primary` | `#141413` | Filled button background |
| `--color-btn-primary-hover` | `#1f1e1d` | Filled button hover |
| `--color-btn-primary-text` | `#ffffff` | Filled button text |

## Guidance

- Use warm neutral surfaces generously; avoid harsh pure-white layouts.
- Keep terra cotta small and intentional so the theme stays understated.
- Use dark filled buttons for the clearest primary action.
