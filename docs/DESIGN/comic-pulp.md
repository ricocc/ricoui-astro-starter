# Comic Pulp Theme

Comic Pulp is a soft pulp comic color theme. It uses pale paper backgrounds, grape-lavender structure, lemon yellow action, and graphic ink contrast.

## Runtime

- Theme id: `comic-pulp`
- Applied with: `<html data-theme="comic-pulp">`
- Stored in: `localStorage.theme_id`
- Dark mode is layered with: `<html class="dark" data-theme="comic-pulp">`

## Light Tokens

| Token | Value | Usage |
| --- | --- | --- |
| `--color-primary` | `#6b5fb5` | Brand color, links, icons |
| `--color-primary-strong` | `#3f357d` | Hover and emphasis |
| `--color-primary-light` | `#c8c1f0` | Soft accents and brand surfaces |
| `--color-accent` | `#f2dc4d` | Lemon yellow action and emphasis |
| `--color-bg-primary` | `#fbf7ea` | Page canvas |
| `--color-bg-secondary` | `#fffdf5` | Cards and panels |
| `--color-text-primary` | `#241c3d` | Heading color |
| `--color-text-secondary` | `#473f67` | Body text |
| `--color-text-tertiary` | `#806f4b` | Muted metadata |
| `--color-btn-primary` | `#f2dc4d` | Filled button background |
| `--color-btn-primary-hover` | `#f6cc5f` | Filled button hover |
| `--color-btn-primary-text` | `#241c3d` | Filled button text |

## Guidance

- Keep the page background pale so the comic accents have room to breathe.
- Use yellow for action and emphasis, balanced by grape text instead of pure black everywhere.
- Let the theme feel graphic through color contrast without turning the whole page saturated.
