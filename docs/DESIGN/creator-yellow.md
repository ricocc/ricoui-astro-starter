# Creator Yellow Theme

Creator Yellow is a friendly creator economy color theme. It uses a soft gray canvas, white cards, black text, and bright yellow action.

## Runtime

- Theme id: `creator-yellow`
- Applied with: `<html data-theme="creator-yellow">`
- Stored in: `localStorage.theme_id`
- Dark mode is layered with: `<html class="dark" data-theme="creator-yellow">`

## Light Tokens

| Token | Value | Usage |
| --- | --- | --- |
| `--color-primary` | `#222222` | Brand color, headings, links, icons |
| `--color-primary-strong` | `#000000` | Hover and emphasis |
| `--color-primary-light` | `#717171` | Soft accents and muted UI |
| `--color-accent` | `#ffdd00` | Yellow highlights and CTA support |
| `--color-bg-primary` | `#e5e7eb` | Page canvas |
| `--color-bg-secondary` | `#ffffff` | Cards and panels |
| `--color-text-primary` | `#222222` | Heading color |
| `--color-text-secondary` | `#414141` | Body text |
| `--color-text-tertiary` | `#717171` | Muted metadata |
| `--color-btn-primary` | `#ffdd00` | Filled button background |
| `--color-btn-primary-hover` | `#f7d046` | Filled button hover |
| `--color-btn-primary-text` | `#000000` | Filled button text |

## Guidance

- Let yellow own the main action; avoid adding competing saturated accents.
- Use white cards over the gray canvas to preserve the creator-card feeling.
- Pair yellow with black text for crisp, friendly contrast.
