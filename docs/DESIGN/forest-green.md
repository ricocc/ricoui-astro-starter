# Forest Green Theme

Forest Green is a grounded botanical color theme inspired by `docs/DESIGN/green.md`. It uses ink text, paper surfaces, and a calm green primary.

## Runtime

- Theme id: `forest-green`
- Applied with: `<html data-theme="forest-green">`
- Stored in: `localStorage.theme_id`
- Dark mode is layered with: `<html class="dark" data-theme="forest-green">`

## Light Tokens

| Token | Value | Usage |
| --- | --- | --- |
| `--color-primary` | `#4a7c59` | Brand color, headings, links, icons |
| `--color-primary-strong` | `#356644` | Hover and emphasis |
| `--color-primary-light` | `#8fba98` | Soft accents and focus surfaces |
| `--color-accent` | `#dce9d8` | Badges and quiet highlight surfaces |
| `--color-bg-primary` | `#f7f5f0` | Page canvas |
| `--color-bg-secondary` | `#ffffff` | Cards and panels |
| `--color-text-primary` | `#2f6840` | Heading color |
| `--color-text-secondary` | `#33443a` | Body text |
| `--color-text-tertiary` | `#66736a` | Muted metadata |
| `--color-btn-primary` | `#4a7c59` | Filled button background |
| `--color-btn-primary-hover` | `#356644` | Filled button hover |
| `--color-btn-primary-text` | `#ffffff` | Filled button text |

## Guidance

- Use this theme for reading tools, wellness products, knowledge bases, and calm product sites.
- Keep green as the only meaningful chromatic direction.
- Light and dark variants should both feel natural and low glare.
