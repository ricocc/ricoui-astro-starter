# Midnight Pastel Theme

Midnight Pastel is a dark collaborative color theme. It uses a black canvas, muted pastel cards, violet accents, and warm clay actions.

## Runtime

- Theme id: `midnight-pastel`
- Applied with: `<html data-theme="midnight-pastel">`
- Stored in: `localStorage.theme_id`
- Dark mode is layered with: `<html class="dark" data-theme="midnight-pastel">`

## Light Tokens

| Token | Value | Usage |
| --- | --- | --- |
| `--color-primary` | `#453b60` | Brand color, headings, links, icons |
| `--color-primary-strong` | `#65451d` | Hover and emphasis |
| `--color-primary-light` | `#bbb2ce` | Violet accent surfaces |
| `--color-accent` | `#e4b976` | Ochre highlight |
| `--color-bg-primary` | `#f3f1ec` | Page canvas |
| `--color-bg-secondary` | `#ffffff` | Cards and panels |
| `--color-text-primary` | `#453b60` | Heading color |
| `--color-text-secondary` | `#333333` | Body text |
| `--color-text-tertiary` | `#65451d` | Muted metadata |
| `--color-btn-primary` | `#65451d` | Filled button background |
| `--color-btn-primary-hover` | `#453b60` | Filled button hover |
| `--color-btn-primary-text` | `#ffffff` | Filled button text |

## Guidance

- Let the dark base carry the interface, then use pastels as controlled highlights.
- Use clay and ochre for warm actions rather than generic bright colors.
- Avoid sharp neon colors; keep the palette muted and tactile.
