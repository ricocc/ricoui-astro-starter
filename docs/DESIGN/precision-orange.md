# Precision Orange Theme

Precision Orange is a technical neutral color theme. It uses stark gray surfaces, black text, and one precise orange signal.

## Runtime

- Theme id: `precision-orange`
- Applied with: `<html data-theme="precision-orange">`
- Stored in: `localStorage.theme_id`
- Dark mode is layered with: `<html class="dark" data-theme="precision-orange">`

## Light Tokens

| Token | Value | Usage |
| --- | --- | --- |
| `--color-primary` | `#020202` | Brand color, headings, links, icons |
| `--color-primary-strong` | `#3d3a39` | Hover and emphasis |
| `--color-primary-light` | `#b8b3b0` | Soft accents and borders |
| `--color-accent` | `#ef6f2e` | Orange status signal |
| `--color-bg-primary` | `#eeeeee` | Page canvas |
| `--color-bg-secondary` | `#fafafa` | Cards and panels |
| `--color-text-primary` | `#020202` | Heading color |
| `--color-text-secondary` | `#3d3a39` | Body text |
| `--color-text-tertiary` | `#756f6b` | Muted metadata |
| `--color-btn-primary` | `#020202` | Filled button background |
| `--color-btn-primary-hover` | `#3d3a39` | Filled button hover |
| `--color-btn-primary-text` | `#ffffff` | Filled button text |

## Guidance

- Use orange sparingly for new, active, or technical signal states.
- Lean on black, gray, and spacing for hierarchy rather than decoration.
- Keep the palette crisp and utilitarian.
