# Retro Blue Theme

Retro Blue is the default RicoUI Astro Starter theme. It pairs a warm paper canvas with editorial blue headings, soft blue accents, and a small gold highlight.

## Runtime

- Theme id: `retro-blue`
- Applied with: `<html data-theme="retro-blue">`
- Stored in: `localStorage.theme_id`
- Dark mode is layered with: `<html class="dark" data-theme="retro-blue">`

## Light Tokens

| Token | Value | Usage |
| --- | --- | --- |
| `--color-primary` | `#2d6dc3` | Brand color, headings, links, icons |
| `--color-primary-strong` | `#0066ff` | Hover and emphasis |
| `--color-primary-light` | `#8fb9ff` | Soft accents and focus surfaces |
| `--color-accent` | `#fad13b` | Badges and highlights |
| `--color-bg-primary` | `#fdfaf5` | Page canvas |
| `--color-bg-secondary` | `#ffffff` | Cards and panels |
| `--color-text-primary` | `#2d6dc3` | Heading color |
| `--color-text-secondary` | `#3f4a5a` | Body text |
| `--color-text-tertiary` | `#7a6550` | Muted metadata |
| `--color-btn-primary` | `#2d6dc3` | Filled button background |
| `--color-btn-primary-hover` | `#0066ff` | Filled button hover |
| `--color-btn-primary-text` | `#ffffff` | Filled button text |

## Guidance

- Use this theme when you want the original RicoUI personality.
- Keep gold as a restrained secondary accent, not a second primary brand color.
- Preserve the warm canvas in light mode; it is part of the theme identity.
