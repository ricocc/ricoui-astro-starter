# Rico Red Theme

Rico Red is a pragmatic red-accent color theme. It uses a warm gray-white canvas, strong graphite text, and confident red action.

## Runtime

- Theme id: `rico-red`
- Applied with: `<html data-theme="rico-red">`
- Stored in: `localStorage.theme_id`
- Dark mode is layered with: `<html class="dark" data-theme="rico-red">`

## Light Tokens

| Token | Value | Usage |
| --- | --- | --- |
| `--color-primary` | `#d04841` | Brand color, links, active states |
| `--color-primary-strong` | `#a83b35` | Hover and emphasis |
| `--color-primary-light` | `#f2aaa5` | Soft accents |
| `--color-accent` | `#eeebea` | Warm gray highlight surfaces |
| `--color-bg-primary` | `#eeebea` | Page canvas |
| `--color-bg-secondary` | `#ffffff` | Cards and panels |
| `--color-text-primary` | `#181717` | Heading color |
| `--color-text-secondary` | `#2e2d2d` | Body text |
| `--color-text-tertiary` | `#706e6d` | Muted metadata |
| `--color-btn-primary` | `#d04841` | Filled button background |
| `--color-btn-primary-hover` | `#a83b35` | Filled button hover |
| `--color-btn-primary-text` | `#ffffff` | Filled button text |

## Guidance

- Reserve red for primary action or unambiguous active state.
- Use warm gray surfaces to soften the enterprise palette.
- Prefer strong text contrast and simple card surfaces over decoration.
