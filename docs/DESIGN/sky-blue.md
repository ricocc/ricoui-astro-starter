# Sky Blue Theme

Sky Blue is a pale blue-white product color theme. It uses a near-white canvas, sky wash surfaces, midnight ink, and vivid blue focus.

## Runtime

- Theme id: `sky-blue`
- Applied with: `<html data-theme="sky-blue">`
- Stored in: `localStorage.theme_id`
- Dark mode is layered with: `<html class="dark" data-theme="sky-blue">`

## Light Tokens

| Token | Value | Usage |
| --- | --- | --- |
| `--color-primary` | `#145aff` | Brand color, links, focus, actions |
| `--color-primary-strong` | `#0f1f3d` | Hover and emphasis |
| `--color-primary-light` | `#b6cbfd` | Soft accents and hero wash |
| `--color-accent` | `#f0f4fe` | Pale sky surfaces |
| `--color-bg-primary` | `#fcfcfc` | Page canvas |
| `--color-bg-secondary` | `#ffffff` | Cards and panels |
| `--color-text-primary` | `#020520` | Heading color |
| `--color-text-secondary` | `#374151` | Body text |
| `--color-text-tertiary` | `#696a72` | Muted metadata |
| `--color-btn-primary` | `#145aff` | Filled button background |
| `--color-btn-primary-hover` | `#0f1f3d` | Filled button hover |
| `--color-btn-primary-text` | `#ffffff` | Filled button text |

## Guidance

- Use blue as a precise signal, not as an all-over paint color.
- Let near-white and pale sky surfaces provide most of the depth.
- Keep secondary text in blue-gray neutrals for a calm product feel.
